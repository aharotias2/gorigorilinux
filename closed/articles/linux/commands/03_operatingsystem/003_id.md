# id - ユーザのID、グループを確認する
## 構文
: id [ユーザ名]
* `id`: コマンド名です。
* ユーザ名 (任意) : 指定しない場合は現在のユーザが対象となります。
## 例
### 例1 - 自分自身が対象
ユーザ「hoge」でログインしている場合の例です。
```ターミナル
プロンプト$ id
uid=1004(hoge) gid=100(users) groups=100(users)
```
### 例2 - 他のユーザが対象
ユーザ「fuga」でログインしている時に、別のユーザ「hoge」の情報を見たい場合の例です。
```ターミナル
プロンプト$ id hoge
uid=1004(hoge) gid=100(users) groups=100(users)
```
