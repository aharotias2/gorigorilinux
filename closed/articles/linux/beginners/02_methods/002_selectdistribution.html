Linuxの種類 (ディストリビューション) を選ぼう

Linuxというのは「Linux系OS」の意味で使われることがほとんどですが、本来はOSの核となる部分のソフトウェアであるカーネルのことです。OSにはユーザーがコンピューターと対話するための多くのソフトウェアが必要です。それらをまとめて提供されたものをディストリビューションと呼びます。Linuxのディストリビューションは非常に多くの種類があり、選ぶのには迷うかもしれません。基本的に無料なので、自分の気に入るものを選んでいけばいいのだと思います。

ディストリビューションには他のディストリビューションの土台となるような有名なものがいくつかあります。かの有名なUbuntuを含めだいたい以下のように大別されます。

  • RedHat系
  • Debian系
  • Ubuntu系
  • Arch系
  • Linux from Scratch系
  • その他
RedHat系

RedHat系は商用のディストリビューションRedHat Enterprise Linux (RHEL)をベースにした派生ディストリビューションです。

Fedora
Redhatが開発しているディストリビューションのうち、最も先進的で、それ故に不安定と評される、ユーザーの総称は「人柱」。半年に一回リリースで、画面サーバWaylandの導入が早かった。人気は高いです。Linuxの創始者リナス・トーバルズ氏も愛用していたとされます。
CentOS
Fedoraとは対照的にRHELのうち商用ライセンスのあるソフトウェアを除いた構成で、安定性に定評があります。書籍も多く、ネット上にもユーザが多い。主にサーバーとして使われています。無難です。仕事向き。
Debian系

Debian系はDebianをベースとしたディストリビューションの一群です。

Debian
dpkgというパッケージ・依存性管理ツールを導入したことで煩雑になりがちなソフトウェア・ライブラリのインストール作業を大幅に簡略化した功績を称えられている老舗ディストリビューション。ただし玄人向きでインストールは難しいそうです。Debian開発者達自身が、リナスがDebianを使ってくれないのはインストールが難しくてできないからでは？と自虐気味に冗談にしているそうです。多くの人に愛されつつも、創始者のイアン・マードックが謎の死を遂げたことが近年、大くの人々を驚かせ、悲しませました。
Ubuntu系

UbuntuはもともとDebianの派生ですが、今はUbuntu系と言っていい派生ディストリビューションが沢山あります。例えば、Mac OS Xのようなエレガントな見た目が売りのElementaryOSなどです。インストーラがほぼ共通なのでどれか一つインストールしたことがあれば、簡単にインストールできます。

Ubuntu
Ubuntu、独自デスクトップ環境Unityを搭載してからは、目新しくユニークな外見とありえないほど重い動作が話題となり、雑誌、書籍が数多く出版されました。ファンでも「Unity糞重い」とよく言っており、Windows XP機の再利用にはちょっと向いていません。数年前までは非常に野心的でしたが、最近はUnityや、モバイルを捨て、大手デスクトップ環境GNOMEをデフォルトにし、休戦状態の感があります。しかしパッケージ管理aptが使い易いので人気は高く、ネット上のLinuxの情報はUbuntuがかなりの割り合いを占めます。サーバー用途でも人気はあります。趣味にも仕事にも。
Xubuntu
Ubuntuのデスクトップ環境を軽量なXfceにしたものです。
Lubuntu
Ubuntuのデスクトップ環境を軽量なLXQTにしたものです。
Kubuntu
Ubuntuのデスクトップ環境をGNOMEと並ぶ重厚長大系DEのKDEにしたです。
ElementaryOS
Mac OS Xのようなオシャレな見た目が売りの独自デスクトップ環境搭載のUbuntu系ディストリビューションです。趣味に良いのでは。有料です。ダウンロード時に支払いを求められます。
Linux Mint
Ubuntu派生でありながら、Ubuntuを凌ぐほどの人気を誇ります。デスクトップ環境がGTK系ながらもGNOMEよりもWindowsに似ていて親しみやすいためと思われます。たしかCinnamonかMATEです。家庭用に良さそうです。
Zorin OS
見た目がガンプラっぽかったのですが、最近はWindows10似になっています。有料です。ダウンロード時に支払いを求められます。インストール失敗するともう一回支払わないといけないので注意です。
Pinguy OS
普通のユーザーが、普通に使えることを目標にしているOSです。操作性に癖がなく、比較的、OSを意識せずに使えると思います。
Voyager
見た目がおしゃれなOSです。Mac寄り。フランス産です。

他にも沢山ありますが、DistroWatchというサイトなどを見るとより正確かつ客観的な情報を入手できます。

Arch系

Archはローリングリリースという言葉を流行らせた先進的なディストリビューションで、派生としてはManjaroが有名です。

Arch Linux
一定期間ごとに最新版をリリースし続けるというローリングリリース・モデルを採用した最初の有名なディストリビューションで、使ったことがない人でも一目置く存在と言えましょう。何故ならカスタマイズ性が売りなのでArchWikiにはフリーソフトウェアの情報満載で非常に役に立つからです。
Manjaro
私のようにArchのせわしないローリングリリースに付いて行けないというユーザのために安定性を重視してカスタマイズされた比較的軽量のLinuxディストリビューションです。
Antergos Linux
Archの設定作業を簡略化して導入しやすくしたディストリビューションです。
Linux from Scratch系

これはLinux OSの全てのソフトウェアをソースコードからコンパイル、インストールしてディストリビューション相当のシステムを構築しようという試みですが、Linux from Scratchのソースをベースにしたディストリビューションも作られているので一応挙げておきます。あまり詳しくは知りません。

その他
Slackware Linux
現存する中では最古のディストリビューションとして有名です。昔はインストールが大変だったそうですが、今ではインストールにDVD一枚で足りますし、最初から必要そうなソフト、ライブラリがだいたい入っているので環境構築は楽です。普通は最初から入っていないGCCやEmacsが最初から入っているのは助かります。
Gentoo Linux
極限までカスタマイズ性を追求してほとんど全てをソースコードから構築するディストリビューションだそうです。男は黙ってGentoo、という方もいます。笑って聞き流しましょう。
Puppy Linux
軽量のディストリビューションで、USBメモリにインストールして起動することができることで、Windowsを潰さずに手軽にLinuxを始めたい人や、困ったときのためのレスキューLinuxとして活用されています。可愛らしい子犬のアイコンが印象的ですが、この子犬ちゃんはただ可愛いだけではなく、狙った獲物に喰らいつく、ハンターの狩猟本能がギラついた瞬間の表情をしています。その辺がこのディストリビューションのちょっと普通じゃないマニアックさを表現していると思われます。しかし基本的には初心者にとっつきやすく、勉強にもなるディストリビューションです。
openSUSE
ドイツで開発されたディストリビューションです。独自のパッケージ管理システムを持っており、そのため独自系に分類しました。厳密にはOpenでない商用のディストリビューションSUSE系になります。オープンスーゼと読みます。
Vine Linux
2000年ごろ、Linuxが一般にもアピールするようになってきたあたりで日本語対応をしたディストリビューションとして、可愛らしいブドウのロゴマークとともに多くのパソコン好きの印象に残ったディストリビューションです。あのころのLinuxは、ちょっとMacユーザには厳しかったです。今と違って、MacはUnixベースではなかったので…。今はもっと良くなっていると思います。
Solus
デスクトップ環境Budgieがデフォルトのディストリビューションです。GNOME3とWindows10の中間のような見た目です。

まだまだ、数えきれないほどのLinuxディストリビューションが存在しますが割愛し、また別の記事で取り上げたいと思います。
