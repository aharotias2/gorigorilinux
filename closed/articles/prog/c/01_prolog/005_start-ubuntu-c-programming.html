初めてのCプログラム (Ubuntu編)

Ubuntu (ウブントゥ) はLinuxをベースとした代表的なディストリビューションでありOSです。今回はこのUbuntuを使ってCプログラミングを始めていきたいと思います。

テキストエディタでC言語プログラムを作成する

Ubuntuの画面の左下にあるアイコンをクリックすると、アプリケーションの一覧が表示されます。

アプリケーションランチャ画面になるので、ここで「テキストエディタ」をクリックします。

テキストエディタが開きました。

ではここにC言語でプログラムを書いていきます。

#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}

これを「hello.c」としてデスクトップに保存します。

C言語コンパイラーとして「gcc」をインストールする

次に、ターミナル (コマンドプロンプトのようなもの) を開きます。Ctrl + Alt + tで開きます。

「gcc」とタイプしてエンターを押すと、gccがまだインストールされていないので「sudo apt intall gcc」というコマンドを打つようにアドバイスがあるのでそれに従って打ちます。

あなた@あなたのホスト:~$ gcc --version [Enter]

Command 'gcc' not found, but can be installed with:

sudo apt install gcc

あなた@あなたのホスト:~$ sudo apt install gcc [Enter]

すると色々出力され、最後までいくとgccがインストールされます。「gcc --version」と打って以下のような出力があれば成功です。

あなた@あなたのホスト:~$ gcc --version [Enter]
gcc (Ubuntu 7.3.0-27ubuntu1~18.04) 7.3.0
Copyright (C) 2017 Free Software Foundation, Inc.
This is free software; see the source for copying conditions.  There is NO
warranty; not even for MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.

あなた@あなたのホスト:~$ cd Desktop [Enter]
あなた@あなたのホスト:~/Desktop$ gcc -o hello hello.c [Enter]

その後、「gcc -o hello hello.c」と打つと先ほど作ったプログラムがコンパイルされます。

ここでも、何も出力されなければ成功です。された場合、恐らくそれはエラーです。直さなきゃいけません。あとでデバッグしましょう!

参考までにですが、世間のUbuntu通によりますとgccを直接インストールせず、関連パッケージも一緒にインストールしてくれる「build-essential」をインストールした方が良いようです。私はその根拠がよく分からなかったので、今回はターミナルのアドバイスを採用してgccを直接インストールすることにしました。多分あとで「make」をインストールすることになると思います。

C言語をコンパイルする + できたプログラムを実行する

では実際に今作ったプログラムを実行してみます。「hello」を実行するのですが、Linux (というよりbash) では環境変数PATHに登録されたディレクトリにあるファイルしか実行できませんので、その他のファイルはディレクトリまで指定して実行する必要があります。普通は実行ファイルがあるディレクトリに移動し、その場所を意味する「./」というパスを付けて実行します。すると以下のようになります。

あなた@あなたのホスト:~$ cd Desktop [Enter]
あなた@あなたのホスト:~/Desktop$ ./hello [Enter]
Hello, world!

はい、これで初めてのC言語プログラムはできました。できましたか？ それはそれは、おめでとうございます!!
