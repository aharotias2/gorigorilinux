# cd - 作業ディレクトリを移動する
`cd`コマンドは作業ディクトリを変更するコマンドです。
引数には作業ディレクトリにしたいディレクトリへの相対パス、または絶対パスを指定します。
## 構文
: cd ディレクトリ

## 例
### 相対パス
```ターミナル
プロンプト$ cd Documents [Enter]
```
上の例では、現在の作業ディレクトリ (CWD) にある「Documents」ディレクトリにCWDを変更します。

### 絶対パス
```ターミナル
プロンプト$ cd /home/you/Documents [Enter]
```
上の例では、移動先のディレクトリを絶対パスによって指定しています。
